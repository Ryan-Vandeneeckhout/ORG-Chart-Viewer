CSV Comparison Tool v1.0

Load two CSVs and see every difference — added rows, removed rows, and field-level changes. Runs entirely in-browser.

01
Upload Files
Drop or click to load CSV A (baseline) and CSV B (updated). Both files stay in your browser.
02
Configure the Comparison
Pick the key column that identifies each row (e.g. Position #, Employee ID). Choose which columns to diff and set match options.
03
Card Display Mapping optional
Map your position, name, and description columns so cards show readable labels.
04
Value Filter optional
Pick a column and add values to filter down to specific rows. Multiple values use OR — any match is shown.
05
Run & Review Results
Hit Run Comparison. Cards are colour-coded by type (Changed · Only in A · Only in B · Matched). Search, filter, and export any subset as CSV.
