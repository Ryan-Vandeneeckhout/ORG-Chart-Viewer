CSV Comparison Tool v1.0

Compare two CSV files row-by-row and surface every difference, addition, and removal — no server, no upload, fully in-browser.

01
Upload Files
Drag & drop or click to load CSV A (your baseline) and CSV B (the updated version). Both files stay local — nothing leaves your browser.
02
Configure the Comparison
Pick the key column that uniquely identifies each row in each file (e.g. Position #, Employee ID). Then choose which columns to compare and set your match options — case sensitivity, whitespace trimming, and empty-field handling.
02b
Map Card Display Fields optional
Tell the tool which columns hold the position number, name, and description so result cards show meaningful labels instead of raw key values.
03
Filter by Value optional
Narrow results to specific rows by picking a column and adding one or more values as filter pills. Matching uses OR logic — any row whose value matches at least one pill is shown.
04
Run & Review Results
Hit Run Comparison. Results appear as colour-coded cards (Changed · Only in A · Only in B · Matched) or a flat table. Use the search bar and type filters to zero in on what matters, then export any subset as CSV.
